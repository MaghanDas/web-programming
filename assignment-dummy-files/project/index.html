Game Rules

Welcome to Budapest Metro — Build & Operate 🚇

This is a turn-based puzzle game where you build metro lines and score points.

How to Play

  1. Enter your name on the main menu and press Start.
  2. Draw a card to reveal a station type (A, B, C, D, or Joker).
  3. Build segments by clicking start and end stations.
  4. Each round has 8 cards.
  5. After 4 rounds, your final score is shown.

Tips

  • Use Jokers wisely — they mimic any station type.
  • Don’t make loops — each line must connect new stations.
  • Plan ahead to connect lines efficiently.
